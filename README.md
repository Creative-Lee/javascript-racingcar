## 실행 방법

### 1. Node.js 17버전 이상🙏

프로젝트에 사용된 Promise API readline은 Node 17버전 이상부터 동작합니다.

17버전 이상의 환경에서 실행해주세요!

### 2. 패키지 다운받기

```bash
npm i
```

위 명령어로 필요한 패키지를 다운로드해 주세요!

### 3. 프로젝트 실행

```bash
node ./src/index.js
```

위 명령어를 터미널에 입력해주세요!

### 4. 레이싱 게임

터미널의 요구에 따라 게임 설정을 하고 즐기세요!
