# 미션 - 자동차 경주 게임

## 기능 구현 목록

- [ ] 자동차 이름 입력 받기
  - [ERROR] 자동차 이름이 5자 초과일 때
- [ ] 각 자동차 생성하기
- [ ] 게임 횟수 입력 받기
  - [ERROR] 입력이 0일 때
  - [ERROR] 입력이 숫자가 아닐 때
- [ ] 각 자동차별로 0에서 9사이의 수를 랜덤 수 생성
- [ ] 각 랜덤 수가 4이상인 경우 자동차 포지션 증가
- [ ] 라운드 결과 출력
- [ ] 위 세 작업을 게임 횟수만큼 반복
- [ ] 최대포지션 자동차(들) 찾기
- [ ] 최종 우승 메세지 출력
- [ ] 게임종료
